# Agenda-WebApp
This web application allows users to save their own activities and set reminders, so that they could be accessible from anywhere at any time.
## Requirements
- The app allows the user to sign up and if the account already exists to log in
- The app allows the user to create notes where he can write down all of his activities
- The app allows the user to create labels where he can add his notes for a better organization
- The app allows the user to create reminders that will notify the user on each device is logged in
- Each user has the possibility to create a group that allows multiple users to have their shared agenda
